# AI Resume Screening System

This project uses NLP techniques to compare a resume with a job description
and calculate a match percentage.

## Features
- Supports PDF, DOCX, and text resumes
- Uses TF-IDF and cosine similarity
- Displays matched and missing skills
- Built using free tools only

## Tech Stack
- Python
- Streamlit
- Scikit-learn

## Run Locally
pip install -r requirements.txt
streamlit run app.py
